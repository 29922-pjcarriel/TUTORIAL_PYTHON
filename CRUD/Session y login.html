Session y Login

Autenticación y Gestión de Sesiones Seguras.

Volver a CRUD

1. Introducción a Sesiones

Las sesiones nos permiten mantener el estado del usuario a través de múltiples peticiones HTTP. Es fundamental para crear áreas protegidas dentro de nuestra aplicación.

2. Implementación del Login

Verificaremos las credenciales del usuario y utilizaremos el objeto session de Flask para marcar al usuario como autenticado.

3. Explicación del Código

Manejo de sesiones en Flask:

# Ejemplo de Login y Sesión
from flask import session, redirect, url_for

app.secret_key = 'mi_clave_secreta_super_segura'

@app.route('/login', methods=['POST'])
def login():
    usuario = request.form['usuario']
    clave = request.form['clave']
    
    if usuario == 'admin' and clave == '1234':
        session['usuario'] = usuario
        return redirect(url_for('dashboard'))
    return "Acceso denegado"

@app.route('/logout')
def logout():
    session.pop('usuario', None)
    return redirect(url_for('index'))
