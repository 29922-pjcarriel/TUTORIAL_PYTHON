    Vehiculo - Parte II

    Creación y Listado de Registros.

    Volver a CRUD

    1. Listado de Vehículos (Read)

    En esta sección implementaremos la ruta para consultar todos los vehículos registrados y mostrarlos en una tabla dinámica.

    2. Formulario de Registro (Create)

    Aprenderemos a procesar el formulario de envío (POST) para insertar nuevos datos en la base de datos de manera segura.

    3. Explicación del Código

    Implementación de las rutas de Flask:

    # Rutas para Listar y Crear
@app.route('/vehiculos')
def listar_vehiculos():
    vehiculos = Vehiculo.query.all()
    return render_template('listar.html', vehiculos=vehiculos)

@app.route('/nuevo_vehiculo', methods=['POST'])
def guardar_vehiculo():
    marca = request.form['marca']
    modelo = request.form['modelo']
    placa = request.form['placa']
    
    nuevo = Vehiculo(marca, modelo, placa)
    db.session.add(nuevo)
    db.session.commit()
    return redirect(url_for('listar_vehiculos'))
