Vehiculo - Parte III

Actualización y Eliminación de Registros.

Volver a CRUD

1. Edición de Datos (Update)

Aprenderemos a cargar los datos existentes en un formulario de edición y persistir los cambios realizados por el usuario.

2. Eliminación de Registros (Delete)

Implementación de la funcionalidad para borrar registros de manera definitiva o lógica utilizando confirmaciones de usuario.

3. Explicación del Código

Implementación de las funciones de actualización y borrado:

# Rutas para Actualizar y Borrar
@app.route('/editar_vehiculo/<id>', methods=['POST'])
def editar_vehiculo(id):
    vehiculo = Vehiculo.query.get(id)
    vehiculo.marca = request.form['marca']
    vehiculo.modelo = request.form['modelo']
    db.session.commit()
    return redirect(url_for('listar_vehiculos'))

@app.route('/eliminar_vehiculo/<id>')
def eliminar_vehiculo(id):
    vehiculo = Vehiculo.query.get(id)
    db.session.delete(vehiculo)
    db.session.commit()
    return redirect(url_for('listar_vehiculos'))
