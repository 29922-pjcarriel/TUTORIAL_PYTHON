Arreglos (Colecciones)

Cómo guardar múltiples datos en una sola variable: Listas, Tuplas y Diccionarios.

Contenido
  • Listas (Lists)
  • Tuplas (Tuples)
  • Diccionarios (Dicts)

1. Listas (Lists)

1. Listas (Lists)

Ordenadas y modificables. Se usan corchetes [].

frutas = ["Manzana", "Banana", "Cereza"]
frutas.append("Naranja") # Agrega al final
print(frutas[0]) # Salida: Manzana

2. Tuplas (Tuples)

2. Tuplas (Tuples)

Ordenadas pero INMUTABLES (no se pueden cambiar). Se usan paréntesis ().

coordenadas = (10, 20)
# coordenadas[0] = 15 <-- ESTO ERROR

3. Diccionarios (Dicts)

3. Diccionarios (Dicts)

Colección de pares Clave: Valor. Se usan llaves {}.

alumno = {
  "nombre": "Elena",
  "curso": "Python",
  "nota": 10
}
print(alumno["nombre"])
