1. Listas (Lists) / Arreglos de Datos

1. Listas (Lists)

Las listas en Python son una forma de guardar varios datos dentro de una sola variable. Cada elemento tiene una posición llamada índice (empieza desde 0).

mi_lista = [2, 1, -6, 4] # Índices # 0 1 2 3

Declaración de listas en Python

Para declarar una lista en Python se siguen las mismas normas básicas utilizadas para declarar variables. La diferencia principal es el uso de corchetes [] o la función list().

Dentro de los corchetes se colocan los elementos que formarán la lista. También es posible usar list() pasando los valores como parámetro para crearla dinámicamente.

EjemploLista = [] otraLista = list() EjemploLista = [1, 2, True, "Hola", 5.8] otraLista = list([4, 9, False, "texto"])

Agregar elementos a la lista

También es posible agregar elementos a una lista después de haberla creado. Para ello se utiliza el método append(), que inserta un nuevo valor al final de la lista.

Otra forma de añadir varios valores a la vez es unir listas usando el operador +, lo que permite concatenar los elementos sin llamar varias veces a append().

paises = [] paises.append("Ecuador") paises.append("Perú") paises.append("Colombia")

Resultado

Ecuador
Perú
Colombia

Mostrar elementos de la lista

Para obtener los elementos de una lista usamos print(). Si queremos acceder a un elemento específico debemos indicar su posición (índice).

print(paises)

Salida

Ecuador
Perú
Colombia
0 1 2

Cada elemento puede mostrarse usando su índice:

print(paises[0]) print(paises[2])

Resultado

Ecuador
Colombia
Índices negativos (recorrido completo)

Cuando usamos índices negativos, Python recorre la lista desde el final. El valor -1 corresponde al último elemento, -2 al penúltimo, y así sucesivamente.

print(paises[-3]) print(paises[-2]) print(paises[-1])

Salida en pantalla

Ecuador
Perú
Colombia
Ecuador
Perú
Colombia
-3 -2 -1

cars = ["Ford", "Volvo", "BMW"]

Elementos de la lista en bucle
edades = [20, 41, 6, 18, 23]
for edad in edades:
  print(edad)
Rango (range) con índices

Sirve para recorrer posiciones con for usando índices.

for i in range(len(edades)):
  print(edades[i])

También se puede con while:

indice = 0
while indice < len(edades):
  print(edades[indice])
  indice += 1
Remover elementos de una lista

pop() elimina por índice, y remove() elimina por valor (solo la primera coincidencia).

palabras = ['hola', 'hello', 'ola']
palabras.pop(1)
print(palabras)
['hola', 'ola']
palabras = ['hola', 'hello', 'hello', 'ola']
palabras.remove('hello')
print(palabras)
['hola', 'hello', 'ola']

2. Listas de listas (Matrices / Tablas)

Listas de listas (Matrices)

Una matriz es una lista que contiene otras listas. Para acceder a un valor se usan dos índices: [fila][columna].

Matriz bidimensional:

matriz = [
  [1, 2],
  [4, 5]
]

Recorrer con for anidados:

for fila in matriz:
  for col in fila:
    print(col)

Ejemplo de tabla (como tu imagen):

EjemploTabla = [["Juan","Laura"],[21,32]]
print(EjemploTabla[0][0])
print(EjemploTabla[-1][1])
print(EjemploTabla[0])
Salida: Juan — 32 — ['Juan','Laura']

Recorrer con índices (for + range):

for i in range(len(EjemploTabla)):
  for j in range(len(EjemploTabla[i])):
    print(EjemploTabla[i][j])

Recorrer con while anidados:

fila = 0
while fila < len(EjemploTabla):
  columna = 0
  while columna < len(EjemploTabla[fila]):
    print(EjemploTabla[fila][columna])
    columna += 1
  fila += 1

3. Diccionarios (Dicts) / Listas Asociativas

3. Diccionarios (Dicts)

Un diccionario almacena elementos como pares clave : valor. Se declara con llaves { }.

EjemploDiccionario = {}
otroDiccionario = {
  "nombre": "Alberto",
  "usuario": "alb545"
}
Mostrar elementos del diccionario
EjemploDiccionario = {
  "nombre": "Juan",
  "usuario": "jn123"
}

print(EjemploDiccionario["nombre"])
print(EjemploDiccionario["usuario"])
Juan
jn123
