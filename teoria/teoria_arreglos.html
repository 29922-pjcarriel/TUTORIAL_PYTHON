1. Listas (Lists) / Arreglos de Datos

1.0 Índices en listas

Las listas en Python son una forma de guardar varios datos dentro de una sola variable. Cada elemento tiene una posición llamada índice (empieza desde 0).

mi_lista = [2, 1, -6, 4] print(mi_lista)
Salida en pantalla:

                    


                    
                    


                      

                        
2

                        
1

                        
-6

                        
4

                      


                      

                        0
                        1
                        2
                        3
                      


                      
índice

                    


                  

                


                
1.2 Ejemplo: Lista con varios tipos

                

                  Las listas en Python pueden almacenar elementos de diferentes tipos de datos al mismo tiempo.
                  Esto las hace muy flexibles para guardar información variada dentro de una sola estructura.
                


                  

                    


                      mi_lista = ["Texto", 45, 3.14, True, [1, 2, 3]]

                      # Tipos almacenados:
                      # string | int | float | bool | lista

                    

                  


                  

                    


                      


                        
Texto

                        
45

                        
3.14

                        
True

                        
[1,2,3]


                      


                      

                        Elementos de diferentes tipos dentro de una lista

                  Son listas que almacenan valores de forma secuencial en una sola dimensión.
                  Ejemplo:
                


                
                


                  
                  

                    


                      ciudades = ["Quito", "Guayaquil", "Cuenca"]

                      # Representación conceptual
                      [ Quito ] [ Guayaquil ] [ Cuenca ]
                      0 1 2

1.4 Declaración de listas en Python


                
                

                  Para declarar una lista en Python se siguen las mismas normas básicas
                  utilizadas para declarar variables. La diferencia principal es el uso
                  de corchetes [] o la función list().
                


                

                  Dentro de los corchetes se colocan los elementos que formarán la lista.
                  También es posible usar list() pasando los valores como
                  parámetro para crearla dinámicamente.
                



                
                


                  
                  

                    

                      EjemploLista = []
                      otraLista = list()

                      EjemploLista = [1, 2, True, "Hola", 5.8]
                      otraLista = list([4, 9, False, "texto"])

                      print(EjemploLista)
                      print(otraLista)
                    


                    
                    

                      

                      
                    


                    
                    

                      
Salida en pantalla:

                      

                    


                    
                    


                      
Contenido de EjemploLista


                      

                        
1

                        
2

                        
True

                        
Hola

                        
5.8

                      


                      


                      
Contenido de otraLista

False

                        
texto

                      


                    

                  


                




                


                
1.5 Agregar elementos a la lista


                
                

                  También es posible agregar elementos a una lista después de haberla creado.
                  Para ello se utiliza el método append(), que inserta un nuevo
                  valor al final de la lista.
                


                

                  Otra forma de añadir varios valores a la vez es unir listas usando el operador
                  +, lo que permite concatenar los elementos sin llamar varias veces
                  a append().
                



                
                


                  
                  

                    


                      paises = []
                      paises.append("Ecuador")
                      paises.append("Perú")
                      paises.append("Colombia")

                    

                  


                  
                  

                    


                      
Resultado


                      

                        
Ecuador

                        
Perú

                        
Colombia

                      


                    

                  


                



                
                


                  
                  

                    


                      numeros = []
                      numeros = numeros + [10,5,3]

                    

                  


                  
                  

                    


                      
Lista resultante


                      

                        
10

                        
5

                        
3

                      


                    

                  


                



                


                
1.6 Mostrar elementos de la lista


                
                

                  Para obtener los elementos de una lista usamos print().
                  Si queremos acceder a un elemento específico debemos indicar su
                  posición (índice).
                


                
                


                  
                  

                    


                      print(paises)

                    

                  


                  
                  

                    


                      
Salida


                      

                        
Ecuador

                        
Perú

                        
Colombia

                      


                      

                        0
                        1
                        2
                      


                    

                  


                



                
                

                  Cada elemento puede mostrarse usando su índice:
                


                


                  

                    


                      print(paises[0])
                      print(paises[2])

                    

                  


                  

                    


                      
Resultado

                      Ecuador

                      Colombia

                    

                  


                



                
                


                
1.7 Índices negativos (recorrido completo)
                



                

                  Cuando usamos índices negativos, Python recorre la lista desde el final.
                  El valor -1 corresponde al último elemento,
                  -2 al penúltimo, y así sucesivamente.
                


                


                  
                  

                    


                      print(paises[-3])
                      print(paises[-2])
                      print(paises[-1])

                    

                  


                  
                  

                    


                      
Salida en pantalla


                      Ecuador

                      Perú

                      Colombia

                      


                      

                        
Ecuador

                        
Perú

                        
Colombia

                      


                      

                        -3
                        -2
                        -1
                      


                    

                  


                


                


                
1.8 Longitud de una lista


                

                  Utilizando el método len(), devuelve la longitud de la lista
                  (número de elementos). Ejemplo:
                


                

                  cars = ["Ford", "Volvo", "BMW"]

                  x = len(cars)

                  print(x) # Mostrará: 3 que corresponde al número de elementos de la lista.
                


                

                  3
                



                


                


                
1.9 Elementos de la lista en bucle


                

                  Para obtener o recorrer todos los elementos de una lista de forma simple; en este caso,
                  los ciclos for van accediendo uno por uno a los elementos. Ejemplo:
                


                

                  edades = [20, 41, 6, 18, 23]

                  for edad in edades:
                  print(edad)
                

                

                  
Salida en pantalla:


                  

                    20

                    41

                    6

                    18

                    23
                  

                




                


                
1.10 Rango


                

                  Genera una secuencia de números enteros, usado comúnmente para recorrer índices en bucles de tipo
                  for, mostrando el valor de la posición solicitada:
                


                

                  edades = [20, 41, 6, 18, 23]

                  for i in range(len(edades)):
                  print(edades[i])
                


                

                  

                  
                



                

                  
Salida en pantalla:

                  

                  

                


                

                  Se observa que el ciclo for puede acceder directamente a los elementos y recorrerlos uno
                  por uno,
                  pero también se accede a ellos mediante sus índices; esto nos devolverá el mismo resultado.
                


                

                  Incluso con el bucle while se puede recorrer los elementos de la lista, mediante los
                  índices,
                  para obtener el mismo resultado que el bucle for:
                


                

                  indice = 0

                  while indice < len(edades): print(edades[indice]) indice +=1 


                    

                      

                      
                    



                    

                      
Salida en pantalla:

                      

                      

                    


                    


                    
1.11 Remover elementos de una lista


                    

                      Puedes usar el método .pop() para eliminar un elemento de la lista, según el índice
                      que se indique.
                      Ejemplo:
                    


                    

                      palabras = ['hola', 'hello', 'ola']
                      palabras.pop(1)
                      print(palabras) # Mostrará: ['hola', 'ola']
                    


                    

                      
                      
                    


                    

                      
Salida en pantalla:

                      

                    


                    

                      Como vemos, el elemento ubicado en el índice 1 ("hello") se eliminó de la lista.
                      Pero si no quieres usar los índices, puedes usar el valor del elemento que deseas borrar.
                      Aplicando la función .remove(), se elimina un elemento según el valor que este tenga
                      al interior de la lista.
                      Ejemplo:
                    



                    

                      palabras = ['hola', 'hello', 'hello', 'ola']
                      palabras.remove('hello')
                      print(palabras) # Mostrará: ['hola', 'hello', 'ola']
                    


                    

                      

                      

                      
                    


                    

                      
Salida en pantalla:

                      

                    




                    

                      Se observa, que remove busca el elemento y lo borra, pero solo borrará el primer
                      elemento que encuentre,
                      no todos los que tengan ese valor.

2. Matrices / Tablas (Listas de listas)

                    tablas o estructuras más complejas.

2.3 Listas de listas (Matrices)


                  

                    También llamadas tablas, no son más que matrices o listas que contienen listas.
                  

                  

                    Son utilizadas para almacenar múltiples valores en una única variable y siguen la misma lógica de
                    las listas,
                    pero con más niveles de “anidamiento”.
                  



                  
2.4 Declarar una tabla en Python

                  

                    Como vimos en listas, las tablas se declaran de la misma manera que una lista:
                  


                  

                    EjemploTabla = []
                    otraTabla = [[], []]

                    print(EjemploTabla)
                    print(otraTabla)
                  


                  

                    
                    
                  


                  

                    
Salida en pantalla:

                    

                  


                  

                    Dentro de los corchetes agregamos las listas con sus elementos y así aplicamos los métodos ya vistos
                    para agregar, quitar u obtener elementos.
                  


                  
2.5 Mostrar elementos de la tabla


                  

                    EjemploTabla = [["Juan","Laura"],[21,32]]

                    print(EjemploTabla[0][0])
                    print(EjemploTabla[-1][1])
                    print(EjemploTabla[0]) # Mostrará: ['Juan', 'Laura']
                  


                  

                    
                    
                  


                  

                    
Salida en pantalla:

                    

                  


                  

                    Al tratarse de una matriz, se debe especificar el índice de la fila y de la columna para mostrar el
                    valor del elemento.
                  


                  
2.6 Elementos de la tabla en bucle


                  

                    Con el ciclo for, se accede uno por uno a los elementos de la tabla (fila y columna).
                    Ejemplo:
                  


                  

                    EjemploTabla = [["Juan","Laura"],[21,32]]

                    for fila in EjemploTabla:
                    for columna in fila:
                    print(columna) # Mostrará: Juan - Laura - 21 - 32
                  


                  

                    
                    
                  


                  

                    
Salida en pantalla:

                    

                  


                  

                    Para recorrer con índices, usamos ciclos for anidados con range():
                  


                  

                    EjemploTabla = [["Juan","Laura"],[21,32]]

                    for i in range(len(EjemploTabla)):
                    for j in range(len(EjemploTabla[i])):
                    print(EjemploTabla[i][j])
                  


                  

                    
                    
                  


                  

                    
Salida en pantalla:

                    

                  


                  

                    Igual que en listas, también se puede recorrer una tabla con while usando contadores:
                  


                  

                    EjemploTabla = [["Juan","Laura"],[21,32]]

                    fila = 0
                    while fila < len(EjemploTabla): columna=0 while columna < len(EjemploTabla[fila]):
                      print(EjemploTabla[fila][columna]) columna +=1 fila +=1 


                      

                        
                        
                      


                      

                        
Salida en pantalla:

                        

                      


                      

                        Usando len() obtenemos el tamaño y con range() recorremos las
                        posiciones con índices.
                        Se evidencia que al ser listas con listas, se pueden aplicar los mismos métodos que en arreglos
                        unidimensionales
                        para agregar, buscar y eliminar elementos.
                      



                      


                      
                      
Elementos de la tabla en bucle

                      

                        Con el ciclo for, se accede uno por uno a los elementos de la tabla, recorriendo
                        cada
                        fila y luego cada columna.
                        Ejemplo:
                      


                      

                        EjemploTabla = [["Juan","Laura"],[21,32]]

                        for fila in EjemploTabla:
                        for columna in fila:
                        print(columna) # Mostrará: Juan - Laura - 21 - 32
                      


                      

                        

                        
                      


                      

                        
Salida en pantalla:

                        

                      


                      

                        Para recorrer usando índices con range(), usamos dos bucles: uno para filas y otro
                        para
                        columnas:
                      


                      

                        EjemploTabla = [["Juan","Laura"],[21,32]]

                        for i in range(len(EjemploTabla)):
                        for j in range(len(EjemploTabla[i])):
                        print(EjemploTabla[i][j])
                      


                      

                        

                        
                      


                      

                        
Salida en pantalla:

                        

                      


                      


                      
                      

                        Igual que en listas, se puede usar while para recorrer filas y columnas, con
                        contadores.
                        Ejemplo:
                      


                      

                        EjemploTabla = [["Juan","Laura"],[21,32]]

                        fila = 0
                        while fila < len(EjemploTabla): columna=0 while columna < len(EjemploTabla[fila]):
                          print(EjemploTabla[fila][columna]) columna +=1 fila +=1 


                          

                            

                            
                          


                          

                            
Salida en pantalla:

                            

                          


                          

                            Se evidencia que, al ser listas con listas en su interior, se pueden aplicar los mismos
                            conceptos que en listas:
                            agregar, buscar, eliminar y recorrer elementos, pero manejando índices por fila y columna.
                          




                          

                          
3. Diccionarios (Dicts) / Listas Asociativas


                          

                           


                              


                                
3.0 Listas Asociativas
                                  (Diccionarios)

                                

                                  También llamadas listas con llaves, donde cada llave es un identificador único de una
                                  posición de la lista,
                                  es decir, en lugar de números, se usan llaves que generalmente son strings.
                                


                                

                                  Tienen la misma lógica que las listas, pero nos permiten organizar e identificar de
                                  manera única las posiciones de la lista,
                                  usando las llaves. Un diccionario almacena múltiples elementos en su interior,
                                  incluyendo otros diccionarios o listas,
                                  incluso datos primitivos.
                                


                                


                                
                                
3.1 Declarar un diccionario en Python

                                

                                  Similar a declarar una tabla en Python, pero en lugar de usar corchetes
                                  [], usamos las llaves {}.
                                  Ejemplo:
                                


                                

                                  EjemploDiccionario = {}
                                  otroDiccionario = {
                                  "nombre": "Alberto",
                                  "usuario": "alb545"
                                  }
                                


                                

                                  

                                  
                                


                                

                                  
Salida en pantalla:

                                  

                                


                                

                                  

                                  Una vez es declarado, podemos añadir elementos, quitarlos, obtenerlos, etc.
                                  Como se puede ver, ahora los elementos se almacenan como clave – valor,
                                  siendo “nombre” la clave y “Alberto” el valor asociado a la clave.
                                


                                


                                
                                
3.2 Mostrar elementos del diccionario

                                

                                  En este caso, además del valor en la posición, también contamos con llaves que tienen
                                  información específica del diccionario.
                                  Ejemplo:
                                


                                

                                  EjemploDiccionario = {
                                  "nombre": "Juan",
                                  "usuario": "jn123"
                                  }

                                  print(EjemploDiccionario["nombre"])
                                  print(EjemploDiccionario["usuario"]) # Mostrará: Juan - jn123
                                


                                

                                  

                                  
                                


                                

                                  
Salida en pantalla:

                                  

                                


                                

                                  

                                  Se accede igual que las listas, mediante corchetes, especificando el nombre de la
                                  llave del diccionario para obtener su valor.
                                


                                


                                
                                
3.3 Elementos del diccionario en
                                  bucle


                                

                                  otroDiccionario = {
                                  "nombre": "Alberto",
                                  "edad": "45",
                                  "usuario": "alb545"
                                  }

                                  for llave in otroDiccionario:
                                  print(llave, ": ", otroDiccionario[llave], sep="")
                                  # Mostrará: nombre: Alberto, edad: 45, usuario: alb545
                                


                                

                                  

                                  
                                


                                

                                  
Salida en pantalla:

                                  

                                


                                

                                  

                                  El ciclo for recorre una por una las llaves del diccionario y luego usar
                                  esa llave para mostrar el valor asignado a la misma dentro del diccionario.
                                  Mediante los corchetes, se accede a los valores asociados a las llaves actuales del
                                  diccionario, y con el parámetro sep="", se elimina el espacio adicional.
