Variables y Constantes

Aprende a guardar información en la memoria, cómo nombrar variables correctamente y cómo manejar constantes por convención en Python.

Contenido
  • Variables
  • Constantes
  • Práctica interactiva
  • Video de apoyo

1. Variables

Una variable es un identificador (un nombre) que se usa para almacenar un valor en la memoria. Ese valor puede cambiar durante la ejecución del programa según la lógica que programes.

Idea rápida: piensa en una variable como una “etiqueta” que le pones a un dato para poder usarlo después.

1.1 Declaración de variables

En Python declarar variables es directo: no usas símbolos como $, no declaras el tipo y Python asigna el tipo automáticamente según el valor (tipado dinámico).

Ejemplo: crear variables

# Declaración de variables (Python detecta el tipo automáticamente)
nombre = "Juan"
edad = 20
precio = 15.75
activo = True

print("Nombre:", nombre)
print("Edad:", edad)
print("Total (edad + 5):", edad + 5)
print("¿Está activo?:", activo)
Salida pendiente...

1.2 Reglas para nombrar variables (snake_case)

Para que tu código sea legible y correcto, se recomienda seguir estas reglas:

  • Debe iniciar con una letra o con _
  • No puede iniciar con números
  • No debe tener espacios
  • Es sensible a mayúsculas/minúsculas: Edad y edad son distintas
  • Convención recomendada: snake_case
total_factura
_nombre
2edad
Inicia con número
nombre-usuario
Guion medio “-” no es válido

2. Constantes

Una constante es un valor que no debería cambiar. En Python se usa la convención: escribirlas en MAYÚSCULAS.

Importante: Python sí te deja cambiar una constante, pero hacerlo rompe buenas prácticas.

Ejemplo: constantes por convención

# Constantes (convención: MAYÚSCULAS)
PI = 3.1416
IVA = 0.12
MAX_USUARIOS = 50

precio = 100
cantidad = 2

total = precio * cantidad * (1 + IVA)
print("Total con IVA:", total)
Salida pendiente...

2.1 ¿Cuándo usar constantes?

  • Tasas e impuestos (IVA, ICE, etc.)
  • Valores matemáticos fijos (PI)
  • Límites del sistema (MAX_USUARIOS, INTENTOS_MAX)
  • Configuraciones (URL_API, RUTA_BASE)

Práctica interactiva

Cambia el valor de una variable y observa cómo cambia la salida.

Ejercicio: variable que cambia

mensaje = "Hola"
print("Antes:", mensaje)

mensaje = "Hola, ya cambié"
print("Después:", mensaje)
Salida pendiente...
