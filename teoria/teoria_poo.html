Clase vs Objeto

Una Clase es el "plano" o molde. Un Objeto es lo que construyes con ese molde.

Clase (Molde)

"Coche"

Objetos (Instancias)

"Ferrari Rojo", "Toyota Azul"

Sintaxis en Python

Usamos el método especial __init__ (constructor) para inicializar el objeto.

class Perro:
    def __init__(self, nombre, raza):
        self.nombre = nombre
        self.raza = raza

    def ladrar(self):
        return "¡Guau!"

# Crear objetos (Instanciar)
mi_perro = Perro("Firulais", "Labrador")
print(mi_perro.nombre) # Salida: Firulais
