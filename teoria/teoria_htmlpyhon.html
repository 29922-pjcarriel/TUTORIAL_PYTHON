El desafío de las páginas estáticas

HTML por sí solo es estático. Esto significa que el contenido no cambia automáticamente según el usuario, la base de datos o la lógica del sistema.

En una app real necesitas mostrar cosas como: el nombre del usuario logueado, una lista de productos, resultados de búsqueda, mensajes, etc. Eso no se puede “a mano” editando HTML para cada persona.

Idea clave: Para que una web sea dinámica, el servidor (Python) envía datos y el HTML “renderiza” esos datos.

Flujo: Navegador Flask HTML

1) Navegador

El usuario abre una URL y envía una solicitud (HTTP).

2) Flask

Ejecuta la lógica y prepara datos para la vista.

3) HTML (Jinja2)

La plantilla se renderiza con datos y se devuelve al navegador.

Sintaxis Básica de Jinja2

Tip: En Flask, el backend (Python) no debería escribir HTML manualmente. Lo correcto es renderizar plantillas con render_template().

Ejemplo Completo

Imagina que quieres mostrar una lista de frutas y resaltar si están disponibles o no.

En Python (app.py):

from flask import render_template

@app.route("/frutas")
def frutas():
    items = [
        {"nombre": "Manzana", "disponible": True},
        {"nombre": "Pera", "disponible": False},
        {"nombre": "Uva", "disponible": True}
    ]
    return render_template("lista_frutas.html", lista=items)

Evolución: de HTML estático a Web dinámica

Una mini línea de tiempo para entender cómo “nace” una página dinámica con Flask.

Paso 1
HTML estático

El navegador carga un archivo con contenido fijo. Todos ven lo mismo.

Paso 2
Ruta en Flask

El usuario visita una URL y Flask ejecuta una función (endpoint).

Paso 3
Datos desde Python

El backend prepara información (listas, diccionarios, consultas a BD).

Paso 4
Jinja2 renderiza

La plantilla arma el HTML final con variables y lógica.

Paso 5
Respuesta final

Flask devuelve HTML ya armado al navegador. ¡Web dinámica lista!

HTTP en 5 minutos (lo esencial)

Cuando navegas en una web, tu navegador y el servidor “conversan” usando HTTP. Flask recibe la petición, ejecuta una función y devuelve una respuesta.
