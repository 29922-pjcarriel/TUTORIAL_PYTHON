TEMA 8

Estructuras de Control

Controla el flujo de tu programa: toma decisiones, repite tareas y maneja la lógica de manera eficiente.

Contenido
  • Condicionales (If)
  • Bucle For
  • Bucle While
  • Break & Continue
  • Cuestionario

1. Condicionales (if / elif / else)

Permiten que el programa tome decisiones y ejecute bloques de código diferentes según se cumplan o no ciertas condiciones.

En Python, la indentación (sangría) es obligatoria y define qué código pertenece al bloque condicional.

ejemplo_condicional.py
edad = 20 if edad < 18: print("Eres menor de edad") elif edad == 18: print("¡Felicidades! Acabas de cumplir la mayoría") else: print("Eres adulto, tienes acceso completo")

2. Bucle For

El bucle for en Python se utiliza para iterar sobre una secuencia (como una lista, una tupla, un diccionario o una cadena de texto). Funciona como un iterador "para cada elemento en...".

Iterando un rango

La función range() es muy útil para generar secuencias numéricas.

ejemplo_for.py
# Imprimir tabla del 5 print("--- Tabla del 5 ---") for i in range(1, 4): resultado = 5 * i print(f"5 x {i} = {resultado}") # Iterando una lista frutas = ["Manzana", "Pera", "Uva"] for fruta in frutas: print(f"Fruta: {fruta}")

3. Bucle While

El bucle while repite un bloque de código mientras una condición específica sea verdadera. Es útil cuando no sabes de antemano cuántas veces necesitas iterar.

ejemplo_while.py
contador = 3 while contador > 0: print(f"Cuenta regresiva: {contador}") contador -= 1 print("¡Despegue!")

4. Sentencias de Control (Break & Continue)

A veces necesitas detener un bucle antes de tiempo o saltar una iteración específica.

break

Detiene el bucle por completo y sale de él, continuando con el resto del programa.

continue

Detiene la iteración actual y salta inmediatamente a la siguiente vuelta del bucle.

ejemplo_break_continue.py
# Ejemplo de break for num in range(10): if num == 3: print("Encontré el 3, deteniendo ciclo...") break print(f"Numero: {num}")
