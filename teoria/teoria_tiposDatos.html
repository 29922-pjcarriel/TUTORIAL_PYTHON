Tipos de Datos en Python

Aprende los tipos básicos: int, float, bool, str y None. En cada sección tendrás un ejemplo interactivo con su salida y el tipo de dato.

Contenido
  • ¿Qué es un tipo de dato?
  • int (enteros)
  • float (decimales)
  • bool (True / False)
  • str (texto)
  • None (sin valor)
  • Video de apoyo

1. ¿Qué es un tipo de dato?

Un tipo de dato define qué clase de valor estás manejando (número, texto, verdadero/falso, etc.). Esto importa porque no puedes hacer las mismas operaciones con todo.

En Python, los tipos de datos no se declaran de forma explícita. El lenguaje utiliza tipado dinámico, lo que significa que Python determina automáticamente el tipo de dato según el valor asignado a la variable, sin necesidad de indicarlo manualmente.

Tip clave: usa type() para ver el tipo de cualquier valor.

2. int (enteros)

Guarda números sin decimales. Se usa para conteos, edades, cantidades, etc.

Error típico: pensar que "20" (texto) es igual que 20 (número). No lo es.

Ejemplo int: suma y type()

edad = 20
extra = 5
total = edad + extra

print("edad:", edad, type(edad))
print("extra:", extra, type(extra))
print("total:", total, type(total))
Salida pendiente...

3. float (decimales)

Guarda números con decimales. Útil para promedios, precios, medidas. Pero puede tener imprecisiones por representación binaria.

Ejemplo clásico: 0.1 + 0.2 puede mostrar 0.30000000000000004.

Ejemplo float: precisión y type()

a = 0.1
b = 0.2
suma = a + b

print("a:", a, type(a))
print("b:", b, type(b))
print("suma:", suma, type(suma))
print("¿suma == 0.3?:", suma == 0.3, type(suma == 0.3))
Salida pendiente...

4. bool (True / False)

Representa verdadero o falso. Se usa en condiciones. Un booleano también puede aparecer como resultado de comparaciones.

Dato importante: type(True) es bool, pero True puede comportarse como 1 en sumas.

Ejemplo bool: comparaciones y “trampa” True=1

activo = True
comparacion = (10 > 3)
suma_bool = True + True

print("activo:", activo, type(activo))
print("comparacion (10 > 3):", comparacion, type(comparacion))
print("True + True:", suma_bool, type(suma_bool))
Salida pendiente...

5. str (texto)

Guarda texto. Un string se define con comillas: "Hola" o 'Hola'. Con strings, el signo + concatena (une texto).

Error típico: "10" + "5" no suma: concatena "105".

Ejemplo str: concatenación y type()

a = "10"
b = "5"
concat = a + b

print("a:", a, type(a))
print("b:", b, type(b))
print("a + b:", concat, type(concat))
Salida pendiente...

6. None (sin valor)

None significa “no hay valor”. Su tipo es NoneType. Es útil cuando quieres indicar que algo todavía no existe o está vacío.

Buena práctica: para comprobarlo se usa is None (no == None).

Ejemplo None: type() y comparación correcta

x = None

print("x:", x, type(x))
print("¿x is None?:", x is None, type(x is None))
print("¿x == None?:", x == None, type(x == None))  # funciona, pero NO es lo recomendado
Salida pendiente...
