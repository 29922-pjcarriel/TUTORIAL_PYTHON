MÓDULO 5

Funciones

Las funciones en PHP y Python permiten organizar el código, recibir parámetros y devolver resultados; la diferencia principal está en la sintaxis.

1 Definición y Llamada de Funciones

Ambos lenguajes permiten definir funciones que reciben parámetros y devuelven resultados, pero la sintaxis difiere significativamente.

PHP

Se usa la palabra clave function, llaves {} y return con punto y coma.

function calcularArea($base, $altura) {
    return $base * $altura;
}

$area = calcularArea(5, 4);
echo $area;  // 20
Python

Se usa def, dos puntos : e indentación. No se necesitan llaves.

def calcular_area(base, altura):
    return base * altura

area = calcular_area(5, 4)
print(area)  # 20

2 Sensibilidad a Mayúsculas y Minúsculas

PHP

En PHP, los nombres de funciones no distinguen entre mayúsculas y minúsculas.

// Ambas llamadas funcionan correctamente
$area  = calculararea(5, 4);  // igual funciona
$area2 = CALCULARAREA(2, 3);  // también funciona

PHP es case-insensitive para nombres de funciones.

Python

Los nombres de funciones sí distinguen entre mayúsculas y minúsculas. Se debe llamar exactamente igual como se definió.

# Esto genera un error
# area2 = CALCULAR_AREA(2, 3)  ✗ Error: NameError

Python es case-sensitive. CALCULAR_AREA no es lo mismo que calcular_area.
