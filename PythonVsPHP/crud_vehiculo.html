IMPORTACIÓN LIBRERÍAS / CLASE

PHP

El archivo principal (index.php) carga manualmente las clases necesarias usando require_once.

index.php
<?php

require_once("class.vehiculo.php");

?>
Usa require_once
PYTHON

El archivo principal (app.py) importa módulos y clases usando import y from ... import.

app.py
if __name__ == "__main__":
  app.run(debug=True)

from flask import Flask, request, render_template_string
import mysql.connector
from class_vehiculo import Vehiculo
Usa IMPORT
Render_template_string: Renderiza HTML definido como texto.
  • El HTML no está en archivos .html
  • Equivalente a echo en PHP

ESTRUCTURA BÁSICA HTML (PYTHON)

PYTHON - App.py Gestión de Static

La carpeta static en Flask se utiliza para almacenar archivos estáticos (recursos que no cambian dinámicamente) como Imágenes, CSS y JS.

static/
  fondo.jpg
  selloespe.jpg
app = Flask(__name__)
HTML = """<!DOCTYPE html ...>
...
<!-- NOTA: url_for genera la URL correcta para archivos estáticos -->
<body style="background: url('{{ url_for('static', filename='fondo.jpg') }}') center ...">

...
{{ contenido|safe }}

  • Flask gestiona automáticamente esta carpeta y permite acceder a sus archivos usando la función url_for().
  • Las dobles llaves {{ }} pertenecen al motor de plantillas Jinja2.

PYTHON Usando mysql.connector
def conectar():
  server = "localhost"

Ambos usan los mismos parámetros y UTF-8.

CONSTRUCTOR

Inicializa la conexión a la base de datos.

PHP

}

En el código se define el método constructor __construct, el cual recibe la conexión a la base de datos como parámetro y la asigna a una propiedad privada.

PYTHON
class Vehiculo:
  def __init__(self, cn):

    ...
    self._con = cn

Se define el método __init__, que actúa como constructor. Inicializa los atributos del objeto y asigna la conexión a la variable self._con.

Consulta una tabla y genera dinámicamente un select HTML con opciones desde la BD.

PYTHON
def __get_combo_db(self, ...):
  cursor = self._con.cursor(dictionary=True)
