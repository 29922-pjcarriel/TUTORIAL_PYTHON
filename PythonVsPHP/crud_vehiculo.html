IMPORTACIÓN LIBRERÍAS / CLASE

PHP

El archivo principal (index.php) carga manualmente las clases necesarias usando require_once.

index.php
<?php

require_once("class.vehiculo.php");

?>
Usa require_once
PYTHON

El archivo principal (app.py) importa módulos y clases usando import y from ... import.

app.py
if __name__ == "__main__":
  app.run(debug=True)

from flask import Flask, request, render_template_string
import mysql.connector
from class_vehiculo import Vehiculo
Usa IMPORT
Render_template_string: Renderiza HTML definido como texto.
  • El HTML no está en archivos .html
  • Equivalente a echo en PHP

ESTRUCTURA BÁSICA HTML (PHP)

PHP - Index.php Estructura Tradicional
  • Usa rutas estáticas para cargar recursos (images/fondo.jpg).
  • La inclusión y manejo de archivos se hace de forma manual.
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" ...>
<html xmlns="http://www.w3.org/1999/xhtml" ...>
<head>
  <title>Matriculas Vehículos PARTE I</title>
  <link href="..." rel="stylesheet">
</head>
<body style="background: url('images/fondo.jpg') center 0px/cover no-repeat fixed;">
  <div class="container py-4">
    <img src="images/selloespe.jpg" ...>
  </div>

ESTRUCTURA BÁSICA HTML (PYTHON)

PYTHON - App.py Gestión de Static

La carpeta static en Flask se utiliza para almacenar archivos estáticos (recursos que no cambian dinámicamente) como Imágenes, CSS y JS.

static/
  fondo.jpg
  selloespe.jpg
app = Flask(__name__)
HTML = """<!DOCTYPE html ...>
...
<!-- NOTA: url_for genera la URL correcta para archivos estáticos -->
<body style="background: url('{{ url_for('static', filename='fondo.jpg') }}') center ...">
  <img src="{{ url_for('static', filename='selloespe.jpg') }}" ...>
...
{{ contenido|safe }}
"""
  • Flask gestiona automáticamente esta carpeta y permite acceder a sus archivos usando la función url_for().
  • Las dobles llaves {{ }} pertenecen al motor de plantillas Jinja2.

CREAR LA CONEXIÓN Y OBJETO VEHICULO

PHP
<?php
$db = conectar();
$objetoVehiculo = new vehiculo($db);

Inicia bloque PHP.
Llama a conectar().
Instancia vehiculo pasando la conexión.

PYTHON
@app.route("/", methods=["GET", "POST"])
def index():
  db = conectar()
  objetoVehiculo = Vehiculo(db)

Define ruta con decorador @app.route.
Dentro de la función, conecta e instancia el objeto.

CONEXIÓN A LA BASE DE DATOS

PHP Usando mysqli
function conectar(){
  $server = "localhost";
  $user = "root";
  $pass = "123";
  $db = "matriculacionfinal";
  $port = 3307;
  $c = new mysqli($server, $user, ...);
  $c->set_charset("utf8");
  return $c;
}
PYTHON Usando mysql.connector
def conectar():
  server = "localhost"
  user = "root"
  password = "123"
  database = "matriculacionfinal"
  port = 3307
  c = mysql.connector.connect(host=server, ...)
  c.set_charset_collation("utf8")
  return c
Ambos usan los mismos parámetros y UTF-8.
