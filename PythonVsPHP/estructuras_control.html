Estructuras de Control

CONDICIONAL

IF
PHP usa llaves para definir bloques, mientras que Python usa indentación.
PHP 
$nota = 15;

if ($nota >= 18) {
    echo "Excelente";
} elseif ($nota >= 11) {
    echo "Aprobado";
} else {
    echo "Desaprobado";
}
PYTHON 
nota = 15

if nota >= 18:
    print("Excelente")
elif nota >= 11:
    print("Aprobado")
else:
    print("Desaprobado")

CONDICIONAL

TERNARIO
Ternario Simple
PHP 
$nota = 15;

// Ternario simple
echo ($nota >= 11) ? "Aprobado" : "Desaprobado";
PYTHON 
nota = 15

# Ternario simple
mensaje = "Aprobado" if nota >= 11 else "Desaprobado"
print(mensaje)
Ternario Anidado
PHP 
$nota = 15;

echo ($nota >= 18) ? "Excelente" :
     ($nota >= 11) ? "Aprobado" :
     "Desaprobado";
PYTHON 
nota = 15

mensaje = "Excelente" if nota >= 18 else "Aprobado" if nota >= 11 else "Desaprobado"
print(mensaje)

BUCLE

FOREACH
El for en Python funciona como un foreach, recorriendo directamente los elementos.
Recorrer un arreglo con índice
PHP 
$numeros = array(10, 20, 30);

foreach ($numeros as $indice => $valor) {
    echo $indice . ": " . $valor . "<br>";
}
PYTHON 
numeros = [10, 20, 30]

for indice, valor in enumerate(numeros):
    print(f"{indice}: {valor}")
Recorrer un arreglo asociativo / diccionario
PHP 
$persona = array(
    "nombre" => "Ana",
    "edad" => 22
);

foreach ($persona as $clave => $valor) {
    echo $clave . ": " . $valor . "<br>";
}
PYTHON 
persona = {
    "nombre": "Ana",
    "edad": 22
}

for clave, valor in persona.items():
    print(clave, ":", valor)

BUCLE

FOR
En PHP el for se basa en contadores, mientras que en Python se usa para recorrer rangos o colecciones. En PHP, el for tiene tres partes: inicialización, condición e incremento. En Python, el segundo valor de range es exclusivo, por eso se usa 6 para llegar hasta 5.
PHP 
for ($i = 1; $i <= 5; $i++) {
    echo $i . "<br>";
}
PYTHON 
# range(inicio, fin)
for i in range(1, 6):
    print(i)

BUCLE

WHILE
El while funciona de forma similar en ambos lenguajes, pero PHP usa llaves y Python indentación. "<br>" → salto de línea en HTML. Python agrega el salto de línea automáticamente.
PHP 
$contador = 1;

while ($contador <= 5) {
    echo $contador . "<br>";
    $contador++;
}
PYTHON 
contador = 1

while contador <= 5:
    print(contador)
    contador += 1

BUCLE

DO-WHILE
PHP tiene do while que se ejecuta al menos una vez, mientras que Python no lo incluye de forma nativa. "<br>" → salto de línea en HTML. Python agrega el salto de línea automáticamente.
PHP 
$contador = 10;

do {
    echo $contador . "<br>";
    $contador++;
} while ($contador <= 5);
PYTHON 
contador = 10

while True:
    print(contador)
    contador += 1

    if contador > 5:
        break

Switch

PHP usa switch para múltiples opciones, mientras que Python desde la versión Python 3.10 hay match/case pero antes lo reemplazaba con if-elif o match.
PHP - switch
PHP 
$dia = 3;

switch ($dia) {
    case 1:
        echo "Lunes";
        break;
    case 2:
        echo "Martes";
        break;
    case 3:
        echo "Miércoles";
        break;
    default:
        echo "Día no válido";
}
Python - if/elif (alternativa clásica)
PYTHON 
dia = 3

if dia == 1:
    print("Lunes")
elif dia == 2:
    print("Martes")
elif dia == 3:
    print("Miércoles")
else:
    print("Día no válido")
Python 3.10+ - match/case
PYTHON 
dia = 3

match dia:
    case 1:
        print("Lunes")
    case 2:
        print("Martes")
    case 3:
        print("Miércoles")
    case _:
        print("Día no válido")
