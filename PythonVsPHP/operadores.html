Operadores Lógicos

Los operadores lógicos permiten combinar condiciones en PHP y Python; aunque la sintaxis cambia, el comportamiento es el mismo.

AND
PHP 
$a = true;
$b = false;

if ($a && $b) {
    echo "Verdadero";
} else {
    echo "Falso"; // Esto se imprime
}

if ($a and $a) {
    echo "Verdadero"; // Esto se imprime
}
Python 
a = True
b = False

if a and b:
    print("Verdadero")
else:
    print("Falso")  # Se imprime

if a and a:
    print("Verdadero")  # Se imprime
OR
PHP 
$a = true;
$b = false;

if ($a || $b) {
    echo "Verdadero"; // Se imprime
}

if ($b or $b) {
    echo "Verdadero"; // No se imprime, ambos son false
}
Python 
a = True
b = False

if a or b:
    print("Verdadero")  # Se imprime

if b or b:
    print("Verdadero")  # No se imprime
NOT
PHP 
$a = true;

if (!$a) {
    echo "Verdadero";
} else {
    echo "Falso"; // Se imprime
}
Python 
a = True

if not a:
    print("Verdadero")
else:
    print("Falso")  # Se imprime
