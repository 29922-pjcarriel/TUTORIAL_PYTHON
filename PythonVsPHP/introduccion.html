INTRODUCCIÓN

PHP y Python son lenguajes muy usados en desarrollo web, pero funcionan de manera diferente desde la forma en que se ejecuta el código hasta su sintaxis y estructura.

DIFERENCIAS GENERALES
ENTRE PHP Y PYTHON

PHP

  • Lenguaje diseñado principalmente para el desarrollo web.
  • Se ejecuta principalmente en el servidor web (Apache, Nginx).
  • Sintaxis inspirada en C, Java y Perl; usa símbolos como $.
  • Se integra directamente con HTML.

PYTHON + FLASK

  • Lenguaje de propósito general (web, IA, datos, automatización, etc.).
  • Se ejecuta mediante un intérprete y puede usarse en muchos entornos.
  • Sintaxis simple y legible; usa la indentación para estructurar el código.
  • Usa frameworks como Django o Flask para desarrollo web.

¿Cómo funciona PHP para
DESARROLLO WEB?

Flujo de PHP

  • PHP se ejecuta directamente en el servidor.
  • El servidor procesa el archivo .php.
  • El navegador solo recibe HTML.
<?php
echo "Hola mundo";
?>

Flujo de Python

  • Se usan frameworks como Flask.
  • Se importa Flask y se crea una aplicación.
  • El programador inicia el servidor manualmente.
from flask import Flask

app = Flask(__name__)

@app.route("/")
def inicio():
    return "Hola mundo"

if __name__ == "__main__":
    app.run(debug=True, host="127.0.0.1", port=5000)

DIFERENCIAS DE SINTAXIS

VARIABLES Y TIPOS DE DATOS

PHP y Python usan tipado dinámico: el tipo de dato se asigna automáticamente en tiempo de ejecución.

PHP

<?php
$edad = 18;      // Entero
$precio = 19.99;  // Decimal
$nombre = "Carlos"; // Texto
$activo = true;    // Booleano

$numeros = array(1, 2, 3);  // Arreglo
$persona = [            // Arreglo asociativo
    "nombre" => "Ana",
    "edad" => 22
];

$valor = null;    // Nulo
?>

PYTHON

edad = 18          # Entero
precio = 19.99      # Decimal
nombre = "Carlos"    # Texto
activo = True       # Booleano

numeros = [1, 2, 3]  # Lista
persona = {          # Diccionario
    "nombre": "Ana",
    "edad": 22
}

valor = None         # Nulo
