Programación Orientada a Objetos (POO)

Comparativa de clases, atributos, constructores, métodos, inclusión e instanciación entre PHP y Python.

1 Clases y Archivos

En PHP los nombres de clases no distinguen mayúsculas y minúsculas, mientras que en Python sí.

PHP — class.vehiculo.php 
<?php
class vehiculo{
    // ...
}
Python — vehiculo.py 
class Vehiculo:
        # ...

En PHP los nombres de clases no distinguen mayúsculas y minúsculas, mientras que en Python sí. La convención en Python es usar CamelCase para nombres de clases.

2 Atributos

PHP

Se declaran con modificadores de acceso explícitos (private, public, protected).

private $foto;
private $avaluo;
private $con;
Python

Se definen en __init__. Doble guion bajo (__) indica privado por name mangling.

def __init__(self, cn):
    self.__foto = None
    self.__avaluo = None
    self.__con = cn

3 Constructor

PHP usa __construct implícito, Python usa __init__ y requiere self.

PHP 
function __construct($cn){
    $this->con = $cn;
}

$this se refiere automáticamente al objeto actual.

Python 
def __init__(self, cn):
    self.__con = cn

En Python, todos los métodos necesitan recibir explícitamente la referencia al objeto actual, y esa referencia se llama convencionalmente self.

4 Métodos

Son funciones dentro de una clase. Comparación de visibilidad:

Método Público
PHP 
// Método público: accesible desde fuera de la clase
public function get_list_public() {
    // ...
}
Python 
# Método público
def get_list(self):
        # ...
Método Protegido
PHP 
// Método protegido: accesible solo dentro
// de la clase y sus subclases
protected function get_list_protected() {
    // ...
}
Python 
# Método "protegido" (convención, se puede usar
# pero indica que es interno)
def _get_list_protected(self):
        # ...
Método Privado
PHP 
// Método privado: accesible solo dentro de esta clase
private function get_list_private() {
    // ...
}
Python 
# Método "privado" (name mangling, accesible
# solo dentro de la clase)
def __get_list_private(self):
        # ...

6 Instanciación de Objetos

PHP usa la palabra clave new, Python no la necesita.

PHP 
$cn = conectar();
$v  = new vehiculo($cn);

Se usa new para crear una instancia de la clase.

Python 
cn = conectar()
v  = Vehiculo(cn)

No se necesita new. Simplemente se llama a la clase como una función.
