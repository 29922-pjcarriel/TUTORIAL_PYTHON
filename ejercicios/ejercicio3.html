Factorial

Cálculo de factorial usando estructuras de control.

Enunciado

En este ejercicio se desarrolla un programa en Flask que permite al usuario ingresar un número entero y calcular su factorial.

Objetivo: comprender cómo Flask procesa formularios con GET y POST, valida datos y muestra resultados dinámicamente.

Explicación del código (paso a paso)

0) Idea general (GET vs POST)
  • GET: se muestra el formulario para ingresar el número.
  • POST: el usuario presiona Calcular y Flask procesa el factorial.
  • Flask identifica el método con request.method.
1) Entrada del usuario

El número ingresado en el formulario llega al servidor mediante request.form. 

numero = int(request.form.get("ingresoNumero", 0))
2) Validación del dato
  • Se verifica que el valor sea un entero.
  • Se limita el rango permitido (0 a 10) para evitar números muy grandes.
  • Si ocurre un error, se muestra un mensaje al usuario.
3) Proceso / Cálculo del factorial

El factorial se calcula multiplicando todos los números desde 1 hasta el número ingresado. 

total = 1
for i in range(1, n + 1):
    total *= i
4) Salida / Resultados
  • Si el cálculo es correcto, se muestra el factorial en pantalla.
  • Si hay un error, se presenta un mensaje indicando el problema.

8) Archivos del proyecto (Paso importante)

Para ejecutar este ejercicio en Flask (factorial lado servidor), crea esta estructura: 

01.Clase3_Factorial/
│
├── app.py
│
└── templates/
    ├── index.html
    └── resultado.html

Ahora crea cada archivo y copia el código.

Archivo: app.py (copiar y pegar)
from flask import Flask, render_template, request

app = Flask(__name__)

def calcular_factorial(n: int) -> int:
    total = 1
    for i in range(1, n + 1):
        total *= i
    return total

# ✅ Acepta "/" y cualquier subruta que el lanzador invente
@app.route("/", methods=["GET", "POST"])
@app.route("/<path:subruta>", methods=["GET", "POST"])
def index(subruta=None):
    if request.method == "POST":
        try:
            numero = int(request.form.get("ingresoNumero", 0))
            resultado = calcular_factorial(numero)
            return render_template("resultado.html", numero=numero, resultado=resultado)
        except Exception:
            return render_template("resultado.html", numero=None, resultado=None)

    return render_template("index.html")

if __name__ == "__main__":
    app.run(debug=True)

Este archivo recibe el número por POST, calcula el factorial con un for y muestra el resultado en otra vista.

9) Ejecutar el proyecto

pip install flask
python app.py

Luego abre en tu navegador:
http://127.0.0.1:5000

Errores comunes
  • No crear la carpeta templates
  • Guardar mal el nombre: debe ser resultado.html
  • No instalar Flask
  • Ejecutar en una carpeta diferente a donde está app.py
