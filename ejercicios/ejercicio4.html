Triángulo de Pascal

Generación del triángulo de Pascal.

Enunciado

En este ejercicio se desarrolla un programa en Python y Flask que permite al usuario ingresar un número de filas y generar el Triángulo de Pascal.

Objetivo: comprender el uso de listas, bucles y lógica matemática, además del manejo de formularios con Flask.

Explicación del código (paso a paso)

0) Idea general (GET vs POST)
  • GET: se muestra el formulario para ingresar el número de filas.
  • POST: el usuario presiona Generar y el servidor construye el triángulo.
  • Flask identifica el método con request.method.
1) Entrada del usuario

El usuario ingresa el número de filas (por ejemplo 5). Este valor se recibe desde el formulario mediante request.form. 

filas = int(request.form.get("numfilas"))
2) Generación del Triángulo de Pascal

El triángulo se genera usando una lista de listas. Cada fila comienza y termina con 1, y los valores intermedios se obtienen sumando los dos valores superiores. 

pascal[i].append(pascal[i-1][j-1] + pascal[i-1][j])
3) Validación del rango

Se controla que el número de filas esté entre 1 y 15. Si no cumple, no se genera el triángulo.

4) Formateo para visualización

Una vez generado, el triángulo se muestra centrado usando texto monoespaciado.

5) Archivos del proyecto (Flask)

Para ejecutar este ejercicio en Flask, crea la siguiente estructura: 

01.Clase4_Pascal/
│
├── app.py
│
└── templates/
    └── index.html

Luego crea cada archivo y copia el código mostrado a continuación.

Archivo: app.py (copiar y pegar)
from flask import Flask, render_template, request

app = Flask(__name__)

# ============================
# LÓGICA DEL TRIÁNGULO DE PASCAL
# ============================

def generar_triangulo_pascal(filas):
    if filas < 1 or filas > 10:
        return []
    
    pascal = []
    
    for i in range(filas):
        pascal.append([])
        pascal[i].append(1)
        
        for j in range(1, i):
            pascal[i].append(pascal[i-1][j-1] + pascal[i-1][j])
        
        if i > 0:
            pascal[i].append(1)
    
    return pascal


def formatear_triangulo_pascal(pascal):
    if not pascal:
        return ""
    
    filas = len(pascal)
    cols = 2 * filas - 1
    centro = filas - 1
    ancho = 4
    
    grid = [['' for _ in range(cols)] for _ in range(filas)]
    
    for r in range(filas):
        for k in range(len(pascal[r])):
            c = centro - r + 2 * k
            grid[r][c] = str(pascal[r][k])
    
    html = '<pre style="font-size:20px; line-height:1.3; text-align:center; font-family:Courier New, monospace;">'
    
    for r in range(filas):
        for c in range(cols):
            if grid[r][c] == '':
                html += ' ' * ancho
            else:
                html += grid[r][c].center(ancho)
        html += '\\n'
    
    html += '</pre>'
    
    return html


# ============================
# RUTA PRINCIPAL
# ============================

@app.route("/", methods=["GET", "POST"])
def index():
    triangulo_html = ""
    filas = ""

    if request.method == "POST":
        try:
            filas = int(request.form.get("filas"))
            pascal = generar_triangulo_pascal(filas)
            triangulo_html = formatear_triangulo_pascal(pascal)
        except:
            triangulo_html = "<p style='color:red;'>Ingrese un número válido (1-10)</p>"

    return render_template("index.html", triangulo=triangulo_html, filas=filas)


if __name__ == "__main__":
    app.run(debug=True)

Este archivo recibe el número de filas mediante POST, genera el triángulo y lo envía al HTML.

Archivo: templates/index.html (copiar y pegar)

    </form>

    <br><br>

    <div>
        {{ triangulo|safe }}
    </div>

</body>
</html>
