Arreglo

Trabajo con arreglos/listas.

Enunciado

Se tiene un diccionario llamado G8 donde:
Clave = Continente (EUROPA, ASIA, AMERICA, etc.)
Valor = una lista de países del G8 o null si ese continente no tiene países del G8.

  • Imprimir la estructura completa del diccionario G8.
  • Recorrer el diccionario e imprimir los países por cada continente.
  • Si el valor es null, mostrar: No tiene países en el G8.

Objetivo: practicar diccionarios (clave/valor), listas y validación de valores nulos.

Explicación del código (paso a paso)

1) Diccionario G8 (clave → valor)

Un diccionario guarda pares clave y valor. Aquí la clave es el continente y el valor es una lista de países (o null). 

G8 = {
  "EUROPA": ["Alemania", "Francia", "Italia", "Reino Unido"],
  "ASIA": ["Rusia", "Japón"],
  "AMERICA": ["USA", "Canada"],
  "AFRICA": null,
  "OCEANIA": null
}
2) Imprimir toda la estructura

Para mostrar el diccionario completo en pantalla se convierte a texto con formato usando JSON.stringify. 

salidaEstructura.textContent = JSON.stringify(G8, null, 2);
3) Recorrer continentes y países

Object.entries(G8) devuelve pares [continente, paises]. Si paises es una lista, se imprime cada país; si es null, se muestra el mensaje. 

for (const [continente, paises] of Object.entries(G8)) {
  if (Array.isArray(paises) && paises.length > 0) {
    // imprimir países
  } else {
    // "No tiene países en el G8"
  }
}

Este ejercicio se ejecuta 100% en frontend con JavaScript. No se usa Jinja ni Flask.
