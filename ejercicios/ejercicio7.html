Archivos del proyecto

01.Clase7_Arreglos/
│
├── app.py
│
└── templates/
    └── index.html

Crea la estructura anterior y copia el código de cada archivo.

Archivo: app.py
from flask import Flask, render_template

app = Flask(__name__)

@app.route("/", methods=["GET"])
@app.route("/<path:subruta>", methods=["GET"])  # por tu lanzador
def index(subruta=None):
    # Equivalentes a tu PHP
    Color = ["BLANCO", "ROJO", "VERDE"]
    C = ["BLANCO", "ROJO", "VERDE"]
    datos = ["Juan", 44, 1500.25]

    ciud = ["PICHINCHA", "CAYAMBE", "RUMIÑAHUI", "PORTOVIEJO"]

    ciudades = [
        ["PICHINCHA", "CAYAMBE", "RUMIÑAHUI"],
        ["GUAYAQUIL", "DAULE"],
        ["CUENCA", "GUALACEO"],
        ["PORTOVIEJO", "MANABI", "ROCAFUERTE", "CHONE", "JIPIJAPA"],
    ]

    return render_template(
        "index.html",
        Color=Color,
        C=C,
        datos=datos,
        ciud=ciud,
        ciudades=ciudades
    )

if __name__ == "__main__":
    app.run(debug=True)

Este archivo crea arreglos y una matriz (lista de listas) y los envía al template con Jinja.

Archivo: templates/index.html
<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="utf-8">
  <title>ARREGLOS</title>
  <meta name="viewport" content="width=device-width, initial-scale=1">
</head>
<body>

  <h2>IMPRIMIR A LA CRIOLLA</h2>

  {% for fila in ciudades %}
    {% for item in fila %}
      {{ item }}&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;
    {% endfor %}
    <br>
  {% endfor %}

  <br><br><br>

  <h2>IMPRIME LA ESTRUCTURA (tipo print_r)</h2>
  <pre>{{ ciudades }}</pre>

</body>
</html>

Aquí se recorre la matriz con dos for (fila e item) y también se imprime completa con <pre>.
