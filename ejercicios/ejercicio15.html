Abstracta

Clases abstractas (concepto y aplicación).

Enunciado

En este ejercicio vamos a practicar clases abstractas en Python usando el módulo abc. Crearemos una clase abstracta llamada Figura que obligará a sus clases hijas (Cuadrado, Rectángulo y Triángulo) a implementar métodos como área y perímetro.

Además, vamos a integrar un formulario web en Flask para elegir el tipo de figura y enviar los lados por POST, y con eso calcular área y perímetro.

Explicación del código (paso a paso)

1) Importaciones principales
from abc import ABC, abstractmethod
from math import sqrt
from typing import Dict, Any
from flask import request, render_template_string
  • ABC y abstractmethod: sirven para definir una clase abstracta y métodos obligatorios.
  • sqrt: raíz cuadrada (se usa en el área del triángulo con la fórmula de Herón).
  • request: permite leer datos enviados por el formulario con POST.
  • render_template_string: devuelve HTML como texto para mostrarlo en detalle.html.
2) ¿Qué es una clase abstracta y para qué sirve?

Una clase abstracta es una clase que sirve como “molde” o “contrato”: define qué métodos deben existir en las clases hijas, pero no obliga a tener una implementación completa.

En este ejercicio, Figura obliga a que cualquier figura tenga métodos como: area(), perimetro(), y getters como get_area().
3) Definir la clase abstracta Figura
class Figura(ABC):

    @staticmethod
    def get_form_html() -> str:
        return "..."  # devuelve el formulario HTML como texto

    @abstractmethod
    def area(self) -> None:
        pass

    @abstractmethod
    def perimetro(self) -> None:
        pass

    @abstractmethod
    def get_area(self) -> float:
        pass

    @abstractmethod
    def get_perimetro(self) -> float:
        pass

    @abstractmethod
    def get_tipo(self) -> str:
        pass
  • Figura(ABC) significa que Figura es abstracta.
  • @abstractmethod marca métodos obligatorios para las clases hijas.
  • get_form_html() devuelve el formulario para escoger la figura y sus lados.
4) Crear las clases hijas (Cuadrado, Rectángulo y Triángulo)

Cada clase hija hereda de Figura y debe implementar los métodos obligatorios. Si no implementa alguno, Python no permitirá crear objetos de esa clase. 

class Cuadrado(Figura):
    def __init__(self, lado1: float):
        self.tipo = "cuadrado"
        self.lado1 = float(lado1)
        self._a = 0.0
        self._p = 0.0

    def area(self) -> None:
        self._a = self.lado1 ** 2

    def perimetro(self) -> None:
        self._p = self.lado1 * 4
Nota: en programación orientada a objetos, es común calcular el área y guardarla en una variable privada (como self._a) y luego devolverla con get_area().
5) Procesar el formulario (POST) y decidir qué figura crear
def process_form(form_data: Dict[str, Any]) -> str:
    tipo = (form_data.get("tipo") or "").strip()

    if not tipo:
        return Figura.get_form_html()

    if tipo == "cuadrado":
        l1 = (form_data.get("l1") or "").strip()
        obj = Cuadrado(float(l1))

    elif tipo == "rectangulo":
        l1 = (form_data.get("l1") or "").strip()
        l2 = (form_data.get("l2") or "").strip()
        obj = Rectangulo(float(l1), float(l2))

    elif tipo == "triangulo":
        l1 = (form_data.get("l1") or "").strip()
        l2 = (form_data.get("l2") or "").strip()
        l3 = (form_data.get("l3") or "").strip()
        obj = Triangulo(float(l1), float(l2), float(l3))
  • Se lee tipo para saber qué figura eligió el usuario.
  • Dependiendo del tipo, se crean objetos: Cuadrado, Rectangulo o Triangulo.
  • Luego se llama a obj.area() y obj.perimetro() para calcular.
6) Calcular y mostrar resultados
obj.area()
        obj.perimetro()

        resultado = f"""
        <p><b>Figura:</b> {obj.get_tipo()}</p>

        <p><b>Área:</b> {obj.get_area():.2f}</p>

        <p><b>Perímetro:</b> {obj.get_perimetro():.2f}</p>
        """

        return Figura.get_form_html() + "<hr>" + resultado

Primero calculamos el área y el perímetro. Luego armamos el resultado en un f-string multilínea para que el código sea más legible y fácil de mantener.

Mostramos primero el formulario para que el usuario pueda recalcular, y debajo mostramos el resultado.

Archivo: AbstractasPOO.py
from flask import Flask, render_template, request
from poo_abstractas.cuadrado import Cuadrado
from poo_abstractas.rectangulo import Rectangulo
from poo_abstractas.triangulo import Triangulo
from poo_abstractas.rombo import Rombo
from poo_abstractas.pentagono import Pentagono

app = Flask(__name__)

@app.route("/", methods=["GET", "POST"])
def index():
    resultado = None

    if request.method == "POST":
        tipo = request.form.get("tipo")
        l1 = float(request.form.get("l1", 0))
        l2 = float(request.form.get("l2", 0))
        l3 = float(request.form.get("l3", 0))

        figura = None

        if tipo == "cuadrado":
            figura = Cuadrado(l1)
        elif tipo == "rectangulo":
            figura = Rectangulo(l1, l2)
        elif tipo == "triangulo":
            figura = Triangulo(l1, l2, l3)
        elif tipo == "rombo":
            figura = Rombo(l1, l2)
        elif tipo == "pentagono":
            figura = Pentagono(l1)

        if figura:
            resultado = {
                "tipo": tipo,
                "area": figura.area(),
                "perimetro": figura.perimetro()
            }

    return render_template("form_figuras.html", resultado=resultado)

if __name__ == "__main__":
    app.run(debug=True)

from .figura_abc import Figura

class Rectangulo(Figura):

    def __init__(self, base, altura):
        self.base = float(base)
        self.altura = float(altura)

    def area(self):
        return self.base * self.altura

    def perimetro(self):
        return 2 * (self.base + self.altura)
Archivo: poo_abstractas/triangulo.py
from .figura_abc import Figura
import math

class Triangulo(Figura):

    def __init__(self, a, b, c):
        self.a = float(a)
        self.b = float(b)
        self.c = float(c)

    def area(self):
        s = (self.a + self.b + self.c) / 2
        try:
            return math.sqrt(s * (s - self.a) * (s - self.b) * (s - self.c))
        except ValueError:
            return 0.0

    def perimetro(self):
        return self.a + self.b + self.c
Archivo: poo_abstractas/rombo.py
from .figura_abc import Figura
import math

class Rombo(Figura):

    def __init__(self, d1, d2):
        self.d1 = float(d1)
        self.d2 = float(d2)

    def area(self):
        return (self.d1 * self.d2) / 2

    def perimetro(self):
        # lado = sqrt((d1/2)^2 + (d2/2)^2)
        lado = math.sqrt((self.d1 / 2) ** 2 + (self.d2 / 2) ** 2)
        return 4 * lado
Archivo: poo_abstractas/pentagono.py
from .figura_abc import Figura
import math

class Pentagono(Figura):

    def __init__(self, lado):
        self.lado = float(lado)

    def area(self):
        # fórmula del pentágono regular
        return (5 * self.lado ** 2) / (4 * math.tan(math.pi / 5))

    def perimetro(self):
        return self.lado * 5
