Ejemplo 02

Índices numéricos + índices asociativos.

Enunciado

Se tiene una lista llamada cantones (lista de listas) y un diccionario llamado capitales.
cantones se usa con índices numéricos (0, 1, 2, ...).
capitales se usa con claves (por ejemplo: "Pichincha").

  • Mostrar la estructura completa del diccionario capitales.
  • Mostrar los cantones recorriendo solo los valores del diccionario.
  • Mostrar la capital de cada provincia (siempre es el elemento [0] de la lista).

Objetivo: practicar listas y diccionarios, entendiendo la diferencia entre índices numéricos y índices asociativos.

Teoría y explicación (paso a paso)

1) Índices numéricos (lista de listas)

Una lista se recorre o se consulta usando posiciones numéricas. Por ejemplo, cantones[0] devuelve la primera lista. 

cantones = [
    ["QUITO", "CAYAMBE", "RUMIÑAHUI"],
    ["GUAYAQUIL", "DAULE"],
    ["CUENCA", "GUALACEO"],
    ["PORTOVIEJO", "MANTA", "ROCAFUERTE", "CHONE", "JIPIJAPA"]
]
2) Índices asociativos (diccionario)

Un diccionario relaciona una clave con un valor. Aquí la clave es la provincia y el valor es la lista de cantones. 

capitales = {
    "Pichincha": cantones[0],
    "Guayas": cantones[1],
    "Azuay": cantones[2],
    "Manabi": cantones[3],
    "Esmeraldas": ["ATACAMES", "TONSUPA", "QUININDE", "MUISNE"]
}
3) ¿Cómo se obtiene la capital?

En este ejercicio, la capital es el primer elemento de la lista, por eso se usa [0]. 

# Ejemplos:
capital_pichincha = capitales["Pichincha"][0]  # "QUITO"
capital_guayas    = capitales["Guayas"][0]     # "GUAYAQUIL"
4) Recorridos importantes del diccionario

En Python, hay 3 recorridos comunes: 

# A) Estructura completa (ver todo el diccionario)
print(capitales)

# B) Solo valores (listas de cantones)
for lista in capitales.values():
    print(lista)

# C) Clave + valor (provincia y su lista)
for provincia, lista in capitales.items():
    print("La capital de", provincia, "es:", lista[0])

Nota importante: cuando usas cantones[0] dentro del diccionario, no estás “copiando”, estás usando esa misma lista como valor de la clave.
