Formulario request.form

Desarrolla un formulario usando mñ©todo POST.

Enunciado

En este ejercicio vamos a construir un formulario de figuras en Flask. El usuario podrá escoger una figura (cuadrado, rectángulo o triángulo) e ingresar sus lados.

Objetivo: practicar GET vs POST, validaciñ³n de entradas y mostrar resultados (área y perímetro).

Explicación del código (paso a paso)

0) Idea general (GET vs POST)
  • GET: el usuario entra a la página â†’ se muestra el formulario vacío.
  • POST: el usuario llena los datos y presiona Calcular â†’ Flask procesa y muestra resultado o errores.
  • En Flask detectamos con request.method.
1) ¿Qué datos se envían?

El formulario envía 4 campos: la figura y los lados l1, l2, l3. Todo llega por request.form porque no hay archivos. 

figura = request.form.get("figura")
l1 = request.form.get("l1")
l2 = request.form.get("l2")
l3 = request.form.get("l3")
2) Guardar lo que escribió el usuario (para no perderlo)

Se usa form_data para que, si hay errores, el formulario se recargue con los valores que el usuario ya escribió. 

form_data = {"figura": figura, "l1": l1, "l2": l2, "l3": l3}
3) Validación + cálculo con process_figure()

La función process_figure(figura, l1, l2, l3) hace TODO lo importante:

  • Verifica que se haya escogido una figura válida.
  • Convierte cada lado a nñºmero y valida que sea positivo (si no, agrega errores).
  • Calcula área y perímetro segñºn la figura.
result, errors = process_figure(figura, l1, l2, l3)
4) ¿Cómo calcula cada figura?
  • Cuadrado: usa solo l1 â†’ área = l1², perímetro = 4·l1.
  • Rectángulo: usa l1 y l2 â†’ área = l1·l2, perímetro = 2·(l1+l2).
  • Triángulo: usa l1, l2, l3 â†’ primero valida desigualdad triangular y luego calcula.
5) Triángulo: validación + área (Fórmula de Herón)

Para triángulo, se valida que los lados sí formen un triángulo: a+b>c, a+c>b y b+c>a. Si es válido, se calcula el área con Herón: 

perimetro = a + b + c
s = perimetro / 2
area = sqrt(s*(s-a)*(s-b)*(s-c))
6) Tipo de triángulo

Si es triángulo válido, también se clasifica como: equilátero (3 iguales), isósceles (2 iguales) o escaleno (todos distintos).

7) Salida final (qué se muestra al usuario)
  • Si hay errores, se muestran en un recuadro rojo (alert).
  • Si hay resultado, se muestran la figura, lados, área, perímetro y (si aplica) tipo de triángulo.
