Procesamiento de formularios

Formulario request.form

Desarrolla un formulario usando método POST.

Enunciado

En este ejercicio vamos a construir un formulario de figuras en Flask. El usuario podrá escoger una figura (cuadrado, rectángulo o triángulo) e ingresar sus lados.

Objetivo: practicar GET vs POST, validación de entradas y mostrar resultados (área y perímetro).

Explicación del código (paso a paso)

0) Idea general (GET vs POST)
  • GET: el usuario entra a la página → se muestra el formulario vacío.
  • POST: el usuario llena los datos y presiona Calcular → Flask procesa y muestra resultado o errores.
  • En Flask detectamos con request.method.
1) ¿Qué datos se envían?

El formulario envía 4 campos: la figura y los lados l1, l2, l3. Todo llega por request.form porque no hay archivos. 

figura = request.form.get("figura")
l1 = request.form.get("l1")
l2 = request.form.get("l2")
l3 = request.form.get("l3")
2) Guardar lo que escribió el usuario (para no perderlo)

Se usa form_data para que, si hay errores, el formulario se recargue con los valores que el usuario ya escribió. 

form_data = {"figura": figura, "l1": l1, "l2": l2, "l3": l3}
3) Validación + cálculo con process_figure()

La función process_figure(figura, l1, l2, l3) hace TODO lo importante:

  • Verifica que se haya escogido una figura válida.
  • Convierte cada lado a número y valida que sea positivo (si no, agrega errores).
  • Calcula área y perímetro según la figura.
result, errors = process_figure(figura, l1, l2, l3)
4) ¿Cómo calcula cada figura?
  • Cuadrado: usa solo l1 → área = l1², perímetro = 4·l1.
  • Rectángulo: usa l1 y l2 → área = l1·l2, perímetro = 2·(l1+l2).
  • Triángulo: usa l1, l2, l3 → primero valida desigualdad triangular y luego calcula.
5) Triángulo: validación + área (Fórmula de Herón)

Para triángulo, se valida que los lados sí formen un triángulo: a+b>c, a+c>b y b+c>a. Si es válido, se calcula el área con Herón: 

perimetro = a + b + c
s = perimetro / 2
area = sqrt(s*(s-a)*(s-b)*(s-c))
6) Tipo de triángulo

Si es triángulo válido, también se clasifica como: equilátero (3 iguales), isósceles (2 iguales) o escaleno (todos distintos).

7) Salida final (qué se muestra al usuario)
  • Si hay errores, se muestran en un recuadro rojo (alert).
  • Si hay resultado, se muestran la figura, lados, área, perímetro y (si aplica) tipo de triángulo.

Archivo: app.py
from flask import Flask, render_template, request
from backend_formulario.geometry import process_figure

app = Flask(__name__)

@app.route("/", methods=["GET", "POST"])
def index():
    result = None
    errors = []
    form_data = {"figura": "", "l1": "", "l2": "", "l3": ""}

    if request.method == "POST":
        figura = request.form.get("figura", "").strip()
        l1 = request.form.get("l1", "").strip()
        l2 = request.form.get("l2", "").strip()
        l3 = request.form.get("l3", "").strip()

        form_data["figura"] = figura
        form_data["l1"] = l1
        form_data["l2"] = l2
        form_data["l3"] = l3

        result, errors = process_figure(figura, l1, l2, l3)

    return render_template("index.html", result=result, errors=errors, form_data=form_data)

if __name__ == "__main__":
    app.run(debug=True)

Este archivo inicia Flask y procesa los datos enviados por el formulario.

Archivo: backend_formulario/geometry.py
import math

def to_positive_float(value_str, field_name, errors):
    if value_str == "":
        return None
    try:
        value = float(value_str)
        if value <= 0:
            errors.append(f"El valor de {field_name} debe ser positivo.")
            return None
        return value
    except ValueError:
        errors.append(f"El valor de {field_name} debe ser numérico.")
        return None

def is_valid_triangle(sides, errors):
    a,b,c = sides
    if a+b<=c or a+c<=b or b+c<=a:
        errors.append("Los lados ingresados no forman un triángulo válido.")
        return False
    return True

def triangle_type(sides):
    a,b,c = sides
    if a==b and b==c:
        return "equilátero"
    elif a==b or a==c or b==c:
        return "isósceles"
    else:
        return "escaleno"

def process_figure(figura, l1_str, l2_str, l3_str):
    errors=[]
    result=None

    if not figura:
        errors.append("Debe seleccionar una figura.")
        return None, errors

    figura=figura.lower()

    l1 = to_positive_float(l1_str,"l1",errors) if l1_str!="" else None
    l2 = to_positive_float(l2_str,"l2",errors) if l2_str!="" else None
    l3 = to_positive_float(l3_str,"l3",errors) if l3_str!="" else None

    if figura=="cuadrado":
        if l1 is None:
            errors.append("Debe ingresar l1.")
        else:
            area=l1*l1
            perimetro=4*l1
            result={"figura":"Cuadrado","lados":[l1],"area":area,"perimetro":perimetro}

    elif figura=="rectangulo":
        if l1 is None or l2 is None:
            errors.append("Debe ingresar l1 y l2.")
        else:
            area=l1*l2
            perimetro=2*(l1+l2)
            result={"figura":"Rectángulo","lados":[l1,l2],"area":area,"perimetro":perimetro}

    elif figura=="triangulo":
        if l1 is None or l2 is None or l3 is None:
            errors.append("Debe ingresar l1, l2 y l3.")
        else:
            lados=[l1,l2,l3]
            if is_valid_triangle(lados,errors):
                perimetro=sum(lados)
                s=perimetro/2
                area=math.sqrt(s*(s-l1)*(s-l2)*(s-l3))
                tipo=triangle_type(lados)
                result={"figura":"Triángulo","lados":lados,"area":area,"perimetro":perimetro,"tipo_triangulo":tipo}

    return result, errors
