Herencia

Ejercicio de herencia en POO.

Enunciado

Implementar un sistema que utilice Herencia para gestionar diferentes tipos de personas en una empresa. Se debe crear una clase base Persona y una clase derivada Empleado que añada atributos de sueldo y cargo, calculando automáticamente beneficios adicionales.

Objetivo: Comprender cómo las clases pueden heredar atributos y métodos de otras, permitiendo la reutilización de código y la creación de jerarquías lógicas.

Explicación del código (paso a paso)

1) Clase Base: Persona

Definimos los atributos comunes como nombre y edad. Todos los empleados son personas, por lo que heredarán estas características. 

class Persona:
    def __init__(self, nombre, edad):
        self.nombre = nombre
        self.edad = edad
2) Clase Derivada: Empleado

La clase Empleado hereda de Persona usando super() y añade el sueldo. 

class Empleado(Persona):
    def __init__(self, nombre, edad, sueldo):
        super().__init__(nombre, edad)
        self.sueldo = sueldo
3) Lógica de Negocio

El programa calcula un bono del 10% basado en el sueldo base del empleado. 

def calcular_total(self):
    return self.sueldo * 1.10

La herencia permite que el Empleado tenga 'nombre' sin haberlo definido explícitamente en su propia clase.

Archivo: HerenciaPOO.py
from flask import Flask, render_template
import sys
import os

# Ajuste de ruta necesario para importar las clases correctamente
sys.path.append(os.path.dirname(os.path.abspath(__file__)))

# Importa las clases derivadas
from poo_herencia.cuadrado import Cuadrado
from poo_herencia.rectangulo import Rectangulo
from poo_herencia.triangulo import Triangulo

app = Flask(__name__)

@app.route('/')
def index():
    # 1. Objeto Cuadrado
    C1 = Cuadrado(4)
    C1.area()
    C1.perimetro()

    # 2. Objeto Rectángulo
    R1 = Rectangulo("rectangulo", 5, 4)
    R1.area()
    R1.perimetro()

    # 3. Objeto Triángulo
    T1 = Triangulo("triangulo", 5, 4, 3)
    T1.area()
    T1.perimetro()

    # Salida para template
    resultados = [
        {'titulo': "DATOS DEL CUADRADO", 'output': C1.get_output()},
        {'titulo': "DATOS DEL RECTANGULO", 'output': R1.get_output()},
        {'titulo': "DATOS DEL TRIANGULO", 'output': T1.get_output()},
    ]

    return render_template('figuras_herencia.html', resultados=resultados)

if __name__ == '__main__':
    app.run(debug=True, port=5002)
Archivo: poo_herencia/figura.py
class Figura:
    """Clase base para figuras geométricas."""

    def __init__(self, dato=None):
        self.__tipo = dato
        self._area = 0.0
        self._perimetro = 0.0

    def get_tipo(self):
        return self.__tipo

    def area(self):
        raise NotImplementedError("El método 'area' debe ser implementado en la clase derivada.")

    def perimetro(self):
        raise NotImplementedError("El método 'perimetro' debe ser implementado en la clase derivada.")

    def get_output(self):
        output = [
            f"TIPO = {self.get_tipo()}",
            f"AREA = {self._area:.2f}",
            f"PERIMETRO = {int(self._perimetro)}"
        ]
        return output
Archivo: poo_herencia/cuadrado.py
from .figura import Figura

class Cuadrado(Figura):
    def __init__(self, l1=0):
        super().__init__("cuadrado")
        self.__lado = l1

    def area(self):
        self._area = self.__lado * self.__lado

    def perimetro(self):
        self._perimetro = self.__lado * 4

    def get_output(self):
        output = super().get_output()
        output.insert(1, f"LADO = {self.__lado}")
        return output
Archivo: poo_herencia/rectangulo.py
from .figura import Figura

class Rectangulo(Figura):
    def __init__(self, tipo, l1, l2):
        super().__init__(tipo)
        self.__lado1 = l1
        self.__lado2 = l2

    def area(self):
        self._area = self.__lado1 * self.__lado2

    def perimetro(self):
        self._perimetro = (self.__lado1 * 2) + (self.__lado2 * 2)

    def get_output(self):
        output = super().get_output()
        output.insert(1, f"LADO 1 = {self.__lado1}")
        output.insert(2, f"LADO 2 = {self.__lado2}")
        return output
Archivo: poo_herencia/triangulo.py
from .figura import Figura
import math

class Triangulo(Figura):
    def __init__(self, tipo, l1, l2, l3):
        super().__init__(tipo)
        self.__lado1 = l1
        self.__lado2 = l2
        self.__lado3 = l3

    def perimetro(self):
        self._perimetro = self.__lado1 + self.__lado2 + self.__lado3

    def area(self):
        self.perimetro()
        sp = self._perimetro / 2
        try:
            self._area = math.sqrt(sp * (sp - self.__lado1) * (sp - self.__lado2) * (sp - self.__lado3))
        except ValueError:
            self._area = 0.0

    def get_output(self):
        output = super().get_output()
        output.insert(1, f"LADO 1 = {self.__lado1}")
        output.insert(2, f"LADO 2 = {self.__lado2}")
        output.insert(3, f"LADO 3 = {self.__lado3}")
        return output
