Herencia

Ejercicio de jerarquía de clases y reutilización de código.

¿Qué es la Herencia en POO?

La Herencia es un pilar fundamental de la POO que permite crear una nueva clase (clase hija) basada en una clase existente (clase padre). La clase hija hereda todos los atributos y métodos de la clase padre, pudiendo además añadir nuevos comportamientos o modificar los existentes.

Enunciado

Implementar una estructura de clases donde exista una clase base llamada FiguraBase que contenga atributos comunes (tipo, área y perímetro). Luego, crear subclases específicas como Cuadrado, Rectángulo y Triángulo que hereden de la base y realicen sus propios cálculos de área y perímetro.

¿Por qué este ejercicio aplica Herencia?

  • Existe una relación "es un": Un cuadrado es una figura.
  • Se utiliza super() para invocar al constructor del padre.
  • Se aplica reutilización de código al definir atributos comunes en una sola clase superior.
  • Permite el polimorfismo al manejar diferentes figuras bajo un mismo tipo base.

Explicación detallada del código (Herencia)

1) Importación de librerías

Se importa math para el cálculo de la raíz cuadrada en la fórmula de Herón y request para procesar los datos del formulario. 

import math
from flask import request
2) Definición de la Superclase FiguraBase

Esta es la clase padre que centraliza los atributos comunes: _tipo, _a (área) y _p (perímetro). Actúa como el molde base para todas las figuras. 

class FiguraBase:
    def __init__(self, tipo=None):
        self._tipo = tipo
        self._a = 0.0
        self._p = 0.0
3) Constructor de la Subclase Cuadrado

La clase Cuadrado hereda de FiguraBase. Usamos super().__init__("cuadrado") para invocar al constructor del padre y pasarle el tipo de figura. 

class Cuadrado(FiguraBase):
    def __init__(self, lado=0):
        super().__init__("cuadrado")
        self.lado = float(lado)
4) Especialización de métodos: Área

Cada clase hija implementa su propia lógica. En el cuadrado, el área se calcula multiplicando el lado por sí mismo y se guarda en el atributo heredado _a. 

def area(self):
    self._a = self.lado * self.lado
5) Especialización de métodos: Perímetro

De igual forma, el perímetro se calcula multiplicando el lado por cuatro y se almacena en _p. 

def perimetro(self):
    self._p = self.lado * 4
6) Subclase Rectangulo

Esta subclase recibe dos parámetros (l1 y l2) y los utiliza para realizar los cálculos correspondientes a su geometría. 

class Rectangulo(FiguraBase):
    def __init__(self, l1, l2):
        super().__init__("rectangulo")
        self.l1 = float(l1)
        self.l2 = float(l2)
7) Subclase Triangulo y Fórmula de Herón

El triángulo hereda de la base y utiliza la fórmula de Herón para el área. Primero calcula el perímetro y luego la raíz cuadrada del semiperímetro. 

def area(self):
    self.perimetro()
    s = self._p / 2
    self._a = math.sqrt(s * (s - self.l1) * (s - self.l2) * (s - self.l3))
8) Función ejecutar() y Selección de Objeto

Dependiendo del tipo recibido por el formulario, se instancia la subclase específica (Cuadrado, Rectángulo o Triángulo), demostrando la flexibilidad de la herencia. 

if tipo == "cuadrado":
    figura = Cuadrado(lado1)
elif tipo == "rectangulo":
    figura = Rectangulo(lado1, lado2)
9) Ejecución Polimórfica

Aunque figura puede ser cualquier objeto hijo, llamamos a los métodos area() y perimetro() de forma genérica. 

figura.area()
figura.perimetro()
10) Presentación de Resultados

Se recuperan los valores finales desde la instancia para mostrarlos en un bloque HTML con formato profesional. 

return f"""
<div class='alert alert-success'>
  <h4>Resultado (Clase: {figura.__class__.__name__})</h4>
  <p><strong>Área:</strong> {figura._a:.2f}</p>
</div>
"""

CÓDIGO COMPLETO DEL EJERCICIO (Clase05_herencia.py)

Implementación completa de la jerarquía de clases con Herencia en Python. 

import math
from flask import Flask, request

app = Flask(__name__)

# =====================================================
# SUPERCLASE (CLASE PADRE)
# =====================================================
class FiguraBase:
    def __init__(self, tipo=None):
        self._tipo = tipo
        self._a = 0.0  # Area
        self._p = 0.0  # Perimetro

    def get_tipo(self):
        return self._tipo

# =====================================================
# SUBCLASES (CLASES HIJAS)
# =====================================================
class Cuadrado(FiguraBase):
    def __init__(self, lado=0):
        super().__init__("cuadrado")
        self.lado = float(lado)

    def area(self):
        self._a = self.lado * self.lado

    def perimetro(self):
        self._p = self.lado * 4

class Rectangulo(FiguraBase):
    def __init__(self, l1, l2):
        super().__init__("rectangulo")
        self.l1 = float(l1)
        self.l2 = float(l2)

    def area(self):
        self._a = self.l1 * self.l2

    def perimetro(self):
        self._p = 2 * (self.l1 + self.l2)

class Triangulo(FiguraBase):
    def __init__(self, l1, l2, l3):
        super().__init__("triangulo")
        self.l1 = float(l1)
        self.l2 = float(l2)
        self.l3 = float(l3)

    def perimetro(self):
        self._p = self.l1 + self.l2 + self.l3

    def area(self):
        self.perimetro()
        s = self._p / 2
        self._a = math.sqrt(s * (s - self.l1) * (s - self.l2) * (s - self.l3))

# =====================================================
# RUTA Y LOGICA DE EJECUCION
# =====================================================
@app.route("/", methods=["GET", "POST"])
def ejecutar():
    if request.method == "POST":
        tipo = request.form.get("tipo")
        lado1 = float(request.form.get("lado1", 0))
        lado2 = float(request.form.get("lado2", 0))
        lado3 = float(request.form.get("lado3", 0))

        if tipo == "cuadrado":
            figura = Cuadrado(lado1)
        elif tipo == "rectangulo":
            figura = Rectangulo(lado1, lado2)
        elif tipo == "triangulo":
            figura = Triangulo(lado1, lado2, lado3)

        figura.area()
        figura.perimetro()

        return f"""
        <div class='alert alert-success'>
          <h2>Resultado (Herencia)</h2>
          <p><strong>Objeto de clase:</strong> {figura.__class__.__name__}</p>
          <p><strong>Área calculada:</strong> {figura._a:.2f}</p>
          <p><strong>Perímetro calculado:</strong> {figura._p:.2f}</p>
          <a href="ejercicio14.html">← Volver</a>
        </div>
        """
    return "Metodo GET no implementado en este fragmento"

if __name__ == "__main__":
    app.run(debug=True)
