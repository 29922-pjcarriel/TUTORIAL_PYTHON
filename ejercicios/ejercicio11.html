Arreglos

Cubo

Trabajo con arreglos 3D (cubo).

¿Qué es un arreglo cúbico?

Un arreglo cúbico es una estructura de datos tridimensional, donde la informaciÃ³n se organiza en tres niveles jerárquicos.

Matemáticamente, un arreglo cúbico se puede representar como: 

  arreglo[ dimensión_1 ][ dimensión_2 ][ dimensión_3 ]

Cada dimensión representa un nivel diferente de agrupación de datos. A diferencia de un arreglo bidimensional (filas y columnas), el arreglo cúbico añade una tercera profundidad.

Enunciado

Desarrollar un programa que utilice arreglos y matrices multidimensionales para representar países, provincias y ciudades. El sistema debe recorrer dichas estructuras y generar dinámicamente tablas HTML que muestren la información organizada.

Objetivo: Comprender el uso de arreglos indexados, arreglos asociativos y recorridos tipo matriz aplicados a un entorno web.

¿Por qué este ejercicio es un arreglo cúbico?

Este ejercicio es cúbico porque la información se organiza en tres dimensiones claramente definidas:

  • Primera dimensión: País
  • Segunda dimensión: Provincia
  • Tercera dimensión: Ciudad (cantón)
  Pais[PAIS][PROVINCIA][CIUDAD]

Cada paí­s contiene varias provincias y cada provincia contiene varias ciudades. Esta relación jerárquica es lo que convierte al ejercicio en un arreglo cúbico.

Estructuras de datos utilizadas

Se utilizan dos tipos de arreglos:

  • Arreglos indexados para las ciudades
  • Arreglos asociativos para países y provincias
  cantones = [
    ["QUITO","CAYAMBE","RUMIÑAHUI"],
    ["GUAYAQUIL","DAULE","SAMBORONDON","BAEZA"]
  ]

  Pais = {
    "ECUADOR": {
      "Pichincha": ["QUITO","CAYAMBE","RUMIÃ‘AHUI"],
      "Guayas": cantones[1]
    }
  }

Recorrido del arreglo cúbico

Para mostrar los datos en forma de tabla, se realiza un recorrido tridimensional:

  1. Se recorre cada país
  2. Dentro de cada país, se recorren las provincias
  3. Dentro de cada provincia, se recorren las ciudades
  for pais in Pais:
      for provincia in Pais[pais]:
          for ciudad in Pais[pais][provincia]:
              procesar ciudad

Este recorrido es característico de los arreglos cúbicos, ya que implica tres niveles de iteración.

Explicación del código (línea por línea)

1) Importación de librerías necesarias

Se importan los componentes requeridos del framework Flask para poder manejar la interacción con el usuario y mostrar resultados dinámicos en una aplicación web.

  • request: permite obtener los datos enviados desde el formulario.
  • render_template_string: permite renderizar HTML directamente desde Python.
from flask import request, render_template_string
2) Definición de la plantilla HTML base

Se define una plantilla HTML que controla dos estados del ejercicio:

  • Estado inicial: muestra únicamente el formulario.
  • Estado de ejecución: muestra el arreglo original y el arreglo con los cubos.

Esta estructura evita que el ejercicio se ejecute automáticamente al cargar la página. 

HTML = """
<h4 class="mb-3 text-center">Ejemplo: Arreglos y Cálculo de Cubos</h4>

<form method="post">
  <input type="text" name="numeros">
  <button type="submit">Ejecutar</button>
</form>


  <p>Arreglo original: </p>
  <p>Cubos calculados: </p>

"""
3) Recepción de datos desde el formulario

Cuando el usuario presiona el botón de ejecutar, los datos ingresados en el formulario son enviados mediante una petición POST. Estos datos se reciben como una cadena de texto. 

numeros = request.form.get("numeros")
4) Conversión de texto a arreglo de enteros

La cadena ingresada por el usuario contiene números separados por comas. Se utiliza una comprensión de listas para:

  • Separar la cadena usando split(",")
  • Convertir cada valor a tipo entero
  • Almacenar los valores en un arreglo (lista)
lista = [int(n) for n in numeros.split(",")]
5) Declaración del arreglo para almacenar los cubos

Se declara un arreglo vacío llamado cubos. En este arreglo se almacenará el resultado del cálculo del cubo de cada número ingresado. 

cubos = []
6) Recorrido del arreglo y cálculo del cubo

Se recorre el arreglo lista utilizando un ciclo for. En cada iteración se calcula el cubo del número actual elevándolo a la potencia de 3.

Matemáticamente: cubo = n * n * n 

for n in lista:
    cubos.append(n ** 3)
7) Envío de resultados a la vista

Finalmente, se enviá el arreglo original y el arreglo con los cubos a la plantilla HTML para ser mostrados al usuario.

Esto permite visualizar claramente la relación entre los datos de entrada y los resultados obtenidos. 

return render_template_string(
    HTML,
    mostrar=True,
    lista=lista,
    cubos=cubos
)
8) Resultado final del ejercicio

El programa muestra:

  • El arreglo original ingresado por el usuario.
  • Un nuevo arreglo que contiene el cubo de cada elemento.

Este ejercicio refuerza conceptos fundamentales como: arreglos indexados, recorridos secuenciales, operaciones matemáticas y transformación de datos mediante programación estructurada.

Código completo del ejercicio (Python)

Analice cuidadosamente el código antes de ejecutar el ejercicio. 

from flask import request, render_template_string

HTML = """
<h4 class="text-center mb-3">TALLER - PAISES</h4>  

"""

def ejecutar():

    cantones = [
        ["QUITO","CAYAMBE","RUMIÑAHUI"],
        ["GUAYAQUIL","DAULE","SAMBORONDON","BAEZA"],
        ["CUENCA","LOJA"],
        ["ROCAFUERTE","CHONE"]
    ]

    Pais = {
        "ECUADOR": {
            "Pichincha": ["QUITO","CAYAMBE","RUMIÑAHUI"],
            "Guayas": cantones[1],
            "Azuay": ["Cuenca"],
            "Manabi": cantones[3],
            "Tunguragua": ["Ambato"]
        }
    }

    if request.method == "GET":
        return render_template_string(HTML, mostrar=False)

    tablas = []

    for pais, info in Pais.items():
        maxColum = max(len(v) for v in info.values())

        html = f"<b>PAIS : {pais}</b><br>"
        html += f"MAXCOLUM: {maxColum}<br>"
        html += f"NUMERO DE PROVINCIAS: {len(info)}<br><br>"

        html += "<table border='1'>"
        html += f"<tr><th colspan='{len(info)}'>{pais}</th></tr><tr>"

        for prov in info.keys():
            html += f"<th>{prov}</th>"
        html += "</tr>"

        for f in range(maxColum):
            html += "<tr>"
            for data in info.values():
                html += f"<td>{data[f] if f < len(data) else ''}</td>"
            html += "</tr>"

        html += "</table><br><br>"
        tablas.append(html)

    return render_template_string(HTML, mostrar=True, tablas=tablas)
